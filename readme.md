## Description

This project is a simple **FPS (First Person Shooter)** game built using Unity. The player can shoot object with pistol.

## Key Features

- 🔫 **Shooting System** 
- 💥 **Muzzle Flash** 
- 🔊 **Sound Effects** 
- 🍾 **Object Destruction** —

## 🕹️ How to Play

### Controls

| Input | Action |
|-------|--------|
| `W A S D` | Move 
| `Mouse Click` | Shoot |
| `Mouse Move` | Aim / look around |

---

## Download & Installation
 
### Clone Repository
 
```bash
git clone <github.com/Kurtz17/SimpleFPS>
```
 
Navigate to the `Game Build` folder after cloning.
 
### How to Run
 
1. Download the `Game(.exe)` folder from the link above
2. Run the file `Robby Shooting Game.exe`
3. Play the game — no installation required
